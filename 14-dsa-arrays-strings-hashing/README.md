# Stage 14: DSA — Arrays, Strings & Hashing 🔢

> *Data structures and algorithms in Rust. We start with the fundamentals: contiguous memory, string manipulation, and hash-based lookups.*

## What You'll Learn

- How arrays and vectors work in memory (contiguous allocation)
- Two-pointer techniques
- Sliding window algorithms
- String manipulation in Rust (UTF-8 aware)
- Hash map internals (hashing, buckets, collision resolution)
- Frequency counting patterns
- Big-O analysis for every operation
- Real-world applications of each technique

## Prerequisites

- Completed [Stage 13: Async Rust](../13-async-rust/) (or at minimum Stages 1–7)
