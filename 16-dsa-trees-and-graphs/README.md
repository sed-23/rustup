# Stage 16: DSA — Trees & Graphs 🌳

> *Hierarchical and networked data. Trees and graphs model everything from file systems to social networks.*

## What You'll Learn

- Binary trees in Rust (using `Option<Box<Node>>`)
- Tree traversals (inorder, preorder, postorder, level-order)
- Binary Search Trees (BST) — insert, search, delete
- Self-balancing trees (AVL tree concepts)
- Heaps and `BinaryHeap` in Rust's standard library
- Graph representations (adjacency list, adjacency matrix)
- BFS and DFS on graphs
- Dijkstra's shortest path algorithm
- Topological sort for DAGs

## Prerequisites

- Completed [Stage 15: DSA — Linked Lists, Stacks & Queues](../15-dsa-linked-lists-stacks-queues/)
