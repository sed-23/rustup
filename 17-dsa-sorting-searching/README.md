# Stage 17: DSA — Sorting, Searching & Recursion 🔍

> *The classics. Every sorting algorithm visualized, every search technique explained, recursion demystified.*

## What You'll Learn

- How Rust's standard sort works (TimSort / pdqsort)
- Bubble, Selection, Insertion sort (and why they're slow)
- Merge Sort — divide and conquer, stable sort
- Quick Sort — partitioning, pivot selection, average O(n log n)
- Heap Sort — in-place, using binary heaps
- Counting Sort, Radix Sort — non-comparison sorts
- Binary Search — classic and variants
- Recursion fundamentals and the call stack
- Backtracking — N-Queens, permutations, subsets
- Divide and conquer paradigm

## Prerequisites

- Completed [Stage 16: DSA — Trees & Graphs](../16-dsa-trees-and-graphs/)
