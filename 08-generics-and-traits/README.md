# Stage 8: Generics & Traits 🧬

> *Write code that works with any type. Traits are Rust's version of interfaces — but more powerful.*

## What You'll Learn

- Generic type parameters in functions, structs, enums, and methods
- What monomorphization is and why generics are zero-cost
- How to define and implement traits
- Trait bounds — constraining generic types
- Default method implementations
- Trait inheritance (supertraits)
- Operator overloading with traits
- Trait objects vs generics (dynamic vs static dispatch)
- Common standard library traits (Display, Debug, Clone, PartialEq, etc.)

## Prerequisites

- Completed [Stage 7: Error Handling](../07-error-handling/)
