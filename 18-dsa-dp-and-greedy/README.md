# Stage 18: DSA — Dynamic Programming & Greedy 🧩

> *The art of breaking big problems into small ones. DP and Greedy are interview favorites — and genuinely useful.*

## What You'll Learn

- What dynamic programming IS (and what it isn't)
- Top-down approach (memoization with recursion)
- Bottom-up approach (tabulation with iteration)
- Classic DP problems: fibonacci, climbing stairs, coin change, knapsack
- 2D DP: longest common subsequence, edit distance, grid paths
- Greedy algorithms: activity selection, Huffman coding
- When to use DP vs Greedy vs Brute Force
- Big-O analysis and space optimization techniques

- Completed [Stage 17: DSA — Sorting & Searching](../17-dsa-sorting-searching/)
