# Stage 2: Rust Fundamentals 🧱

> *The building blocks of every Rust program. Master these and everything else becomes easier.*

## What You'll Learn

- How variables work in Rust (and why they're immutable by default)
- Every data type Rust offers — from integers to tuples
- How to write functions and understand the type system
- Control flow: if/else, loops, match expressions
- How to add comments and write self-documenting code
- Type inference and type annotations
- Constants vs variables

## Prerequisites

- Completed [Stage 1: The Starting Line](../01-the-starting-line/)

## By the End of This Stage

You'll be able to write basic Rust programs using variables, functions, and control flow. You'll understand Rust's type system fundamentals and why immutability by default is a feature, not a limitation.
