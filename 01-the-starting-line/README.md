# Stage 1: The Starting Line 🏁

> *Every expert was once a beginner. This is where your Rust journey begins.*

## What You'll Learn

- Why Rust was created and what problems it solves
- How to install Rust on your machine
- What Cargo is and why it's amazing
- How to write, compile, and run your first Rust program
- How to use the Rust Playground for quick experiments
- How Rust compares to languages you might already know

## Prerequisites

- A computer with internet access
- Basic idea of what programming is (we'll explain everything else)

## By the End of This Stage

You'll have Rust installed, understand why it exists, and have written & run your first program. You'll know how to create projects with Cargo and be comfortable with the basic tooling.
